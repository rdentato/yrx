About YRX

   YRX is a package to ease the creation of lexical scanners, filters and other text transformation tools.

   To create fast and compact tools, YRX transforms a set of patterns into a Deterministic Finite Automaton (DFA) capable of matching those patterns over an input text. This implies that all the patterns are tested in parallel with no need of backtracking. Also, the submatch capture capability of YRX avoid further rescanning of the matching text to extract  parts of it.

   Out of the box, the current version of YRX consist of: 

  • yrl: Lua module to create scanner within a Lua program;
  • yrc: to convert C source code with embedded scanners to a regular C code (similar to the re2c tool);
  • yrf: parse and transform a text file in a fashion similar to AWK or Gema;
  • yrd: Generate a graphic representation of the DFA corresponding to a given set of patterns (requires Graphviz , VCG, aiSee or uGraph to create the image file);

  Other, more specialized, tools can be created using the scripting language Lua, see the page on adding filters for further details.
